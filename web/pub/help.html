URL Parameters Documentation

You can control the initial setup of the mixer by adding parameters to the URL.

Currently supported devices:

- Fireface UCX (Use Port 2)

- Fireface UCX II (Use Port 2)

- Fireface 802 (Use Port 2)

- Fireface UFX III (Use Port R or Port 4)


The base URL is:


Parameter Description Example Action
midi Set connection type to MIDI
midi + auto-connect Enable MIDI and auto-connect to recognized device
midi-input Specify MIDI input device by name
(use quotes for names with spaces)
midi-output Specify MIDI output device by name
(use quotes for names with spaces)
theme Set initial theme
